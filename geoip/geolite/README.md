# GEOIP2（GeoLite2）

GeoLite2数据库是免费的IP地理位置数据库，它可以通过IP地址识别其所在的城市和国家。与MaxMind的GeoIP2数据库相比，准确性较差。

GeoLite2-*.mmdb文件是一个二进制数据库文件，包含了MaxMind GeoLite2所需的全部数据。

GeoLite2的数据库分为：国家(Country)，城市(City)和ASN三个数据库。每周两次更新。
